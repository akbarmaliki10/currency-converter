# Currency Converter (React Practice)

This repository is a personal practice project to learn and experiment with **React.js**. 

## Concepts Explored
- **State Management**: Using `useState` to manage controlled inputs and dropdowns.
- **Performance Optimization**: Using `useMemo` to cache calculations and avoid unnecessary recalculations during renders.
- **Component Extraction**: Breaking down the UI into reusable React components (e.g., `<CurrencySelect />`).
- **Modern UI Styling**: Implementing a responsive, glassmorphism design with plain CSS.

## Getting Started

To run this project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/akbarmaliki10/currency-converter.git
   ```
2. Install the dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
